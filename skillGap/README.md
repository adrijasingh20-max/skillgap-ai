# SkillGap AI

SkillGap AI is a prototype tool that analyzes a student's resume and compares their skills with the requirements of a target career role.

The goal is to help students identify skill gaps and understand what they need to learn for their desired career path.

## Features
- Resume upload (PDF)
- Resume text input
- Skill match analysis
- Missing skill detection
- Market demand visualization
- Personalized learning roadmap

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
- PDF.js

## Purpose
This project helps students understand what skills they need to develop to reach their career goals.
